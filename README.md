It is inspired by [zresume](https://github.com/izuolan/zresume), because zresume is based on Docker, but I know nothing about this technology unfortunately, and it is really very very beautiful, so I make it to a static templete on my own, üòÇ. The design is come from [FREE Resume Template](https://www.behance.net/gallery/15677411/FREE-Resume-Template). ## Installation

0. Emmmmmmmmmn, install Xcode first;
1. Clone the repo with `git clone` command;
2. Open `EFResume.xcworkspace` with Xcode in `core` folder and run.

Or you can run the following command in terminal:

```bash
git clone git@github.com:EyreFree/EFResume.git; cd EFResume/core; open EFResume.xcworkspace
```
